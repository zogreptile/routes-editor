# Routes Editor
Одностраничное приложение, в котором пользователь в интерактивном режиме может создавать на карте маршрут, указывая начальную, конечную и промежуточные точки движения. Для каждой точки маршрута можно посмотреть ее адрес.

### Технологии
- react
- react-yandex-maps
- react-dnd


### Команды
- `yarn start` – запустить проект
- `yarn build` – сборка проекта
